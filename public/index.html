Hello! 👋

Dobrodošao u Gemini Models Explorer

Ova aplikacija je nastala kao ideja — kako sebi da objasnim upotrebu različitih AI modela i krenuo sam od Gemini. Umesto da čitam dokumentaciju stranu po stranu, odlučio sam da napravim živu aplikaciju koja će mi pokazati koji model postoji, šta radi, kako se poziva i kako izgleda API odgovor — sve na jednom mestu. Vremenom je narasla u pravi alat za istraživanje i testiranje.

Svaki model ima sopstvenu karticu sa opisom, karakteristikama, primerima upotrebe i tačnim kodom koji možeš odmah iskoristiti. A u Prompt Testeru možeš poslati pravi zahtev direktno na Gemini API — bez ikakve konfiguracije, samo sa API ključem.

Kako se koristi

  1. 1 Klikni „Preuzmi i Testiraj Modele" — aplikacija povlači aktuelnu listu svih Gemini modela direktno sa Google API-ja.
  2. 2 Na svakoj kartici modela klikni „Detalji" — otvara se modal sa opisom, primerima i gotovim kodom.
  3. 3 Klikni na samu karticu modela da ga izabereš za testiranje — desno se otvara odgovarajući Prompt Tester.
  4. 4 Upiši prompt (ili prevuci fajl), podesi parametre i klikni „Pošalji Zahtev".

API Endpoint Arhitektura

Nije svaki model isti! Različiti modeli koriste različite HTTP rute i formate zahteva — aplikacija to automatski rešava umesto tebe:

Imagen (3 modela) → :predict Synhroni, vraća base64 slike u predictions[]
Veo (6 modela) → :predictLongRunning Asinhrono + polling dok video nije gotov
Lyria (2 modela) → :generateContent Audio modality, vraća audio u inlineData
Embedding modeli → :embedContent Vraća numerički vektor, ne tekst
Sve ostalo (Gemini, Gemma...) → :generateContent Standardni tekst/multimodalni odgovor

Korisni saveti

  • Prevuci sliku, PDF, audio ili video direktno na polje za prompt
  • Svi generisani mediji (slike, audio, video) mogu se odmah preuzeti dugmetom „Sačuvaj"
  • JSON konzola prikazuje tačan request i response za svaki API poziv — korisno za debug
  • Veo video generisanje traje 1–3 minuta — prati progress bar i sačekaj

Pravila korišćenja (Fair Use)

Tekstualni modeli (Gemini, Gemma, Embedding) su besplatni i neograničeni. Međutim, premium modeli koji generišu slike, video ili muziku koriste plaćene API pozive, pa su limitirani kako bi svi korisnici imali fer pristup:

Imagen (generisanje slika)
2 poziva / 2h
Veo (generisanje videa)
2 poziva / 2h
Lyria (generisanje muzike)
2 poziva / 2h
Posle iskorišćenih poziva, pristup se automatski otključava za 2 sata. Broj preostalih poziva videćeš ispod dugmeta „Pošalji Zahtev" kada izabereš premium model. Tekstualni modeli (Gemini, Gemma...) nemaju ograničenja.

Neograničen pristup?

Ova aplikacija je open source! Pokreni je lokalno sa sopstvenim API ključem i dobij neograničen pristup svim modelima — bez ikakvih limita.

1
Kloniraj repo: git clone https://github.com/Newbluewood/Gemini-Models-List-App.git
2
Napravi API ključ na Google AI Studio. Za Imagen/Veo/Lyria, klikni na model u AI Studio i prihvati billing (pay-as-you-go).
3
Kreiraj .env fajl u root-u projekta: GEMINI_API_KEY=tvoj_kljuc_ovde ADMIN_SECRET=tvoj_password
4
Pokreni lokalno: npm install && npm run dev
5
Otključaj — u glavnom interfejsu, klikni na Unlock Limit dugme pored API konfiguracije i upiši svoj password. Password = ADMIN_SECRET iz .env fajla. Status će se promeniti u ∞.
